# AI-Powered Insurance for India’s Gig Economy

## Problem
Gig workers lose income due to rain, heat, and pollution.

## Solution
An AI-based insurance platform that detects disruptions and automatically triggers payouts.

## Features
- Weekly premium model  
- Automatic claim trigger  
- Instant payout
